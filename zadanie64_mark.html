Pseudoelement :first-line

Pseudoelementem :first-line, który obejmuje formatowaniem całą pierwszą linię.

p
{
color:navy;
font-size:12pt;
}
p:first-line
{
color:red;
font-size:20pt;
font-weight:bold;
}

Pseudoelementy :after oraz :before

W Pseudoelementach: :after oraz :before oba elementy służą do umieszczania tekstu przed i za formatowanym elementem strony.

p:before
{
content:"TO SĄ ĆWICZENIA";
font-size:15pt;
font-style:italic;
font-weight:bold;
}
p:after
{
content:"AUTOR Król Lew";
font-size:15pt;
font-style:italic;
font-weight:bold;
}

Pseudoelement :first-letter

Pseudoelement :first-letter pozwala na formatowanie pierwszej litery w bloku tekstu.

H1
{
color:navy;
font-size:30pt;
}
H1:first-letter
{
color:red;
font-size:100pt;
}
